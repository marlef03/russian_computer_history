# russian_computer_history

![logo](/assets/images/readme/logo.png)

## Описание
Веб-сайт для изучения истории советской и российской вычислительной техники

## Возможности
* Изучение полной истории развития советской и российской вычислительной техники в хронологическом порядке (Раздел "Общая история")
* Ознакомление с биографиями личностей, внёсших вклад в развитие отечественной вычислительной техники (Раздел "Значимые личности")
* Знакомство с главными достижениями советских и российских учёных в области вычислительной техники (Раздел "Главные достижения")
* Получение информации о событии, произошедшем в сегодняшний день (Раздел "В этот день...")
* Прохождение теста по материалам общей истории (Раздел "Тесты")
* Отображение прогресса освоения материала и результатов тестов (Раздел "Статистика")
